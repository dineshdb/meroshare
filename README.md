# README

CLI and API for accessing [MeroShare](https://meroshare.cdsc.com.np/)

## Environment Varaibles

- DEMAT_ACCOUNT
- DEMAT_PASSWORD

## Installation

```
# install as global cli
deno install -g --allow-env jsr:@util/meroshare

# add dependency
npx jsr add @util/meroshare
```
